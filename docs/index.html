Proposed Functionality

Synopsis

    Create, edit, update ingredients, recipes.. let the app search for the best priced ingredient, and update the best price found for you across the internet. Search for a recipe containing only ingredients you would like to use, or supply only the available quantities. Search recipes by price range, effort required, time to cook, overall time to be ready to eat, by category, by calories where applicable.. By giving the number of people to eat, the base recipe gets converted accordingly.
